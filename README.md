# PRODIGY_GA_02

## Task
Image Generation with Pre-trained Models

## Description
This project generates AI images from text prompts using the Stable Diffusion pre-trained model. The model converts a user-provided prompt into a realistic image.

## Technologies Used
- Python
- Google Colab
- Diffusers
- PyTorch
- Stable Diffusion

## How to Run
1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Load the Stable Diffusion model.
4. Enter a text prompt.
5. Run the notebook to generate an image.

## Example Prompt
A computer science student learning Generative AI, coding in Python on a laptop, futuristic AI holograms, clean workspace.

## Outcome
Successfully generated an AI image from a text prompt using a pre-trained Stable Diffusion model.
